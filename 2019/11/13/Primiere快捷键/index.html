Primiere快捷键

发表于 2019-11-13 更新于 2020-04-10 分类于 PR ， 剪辑 评论：

1、快速转场 Shift+D
单独视频转场 Ctrl+D
单独音频转场 Ctrl+Shift+D

2、快速波纹剪辑
剪辑视频到时间轴 Q&W
延长剪辑至时间轴 Shift+Q&W

3、匹配帧
在监视器中找到 时间线上对应的帧 F
在时间线上找到 监视器对应的帧 Shift+R

4、快速缩放
快速缩放 \（反斜杠）
隐藏项目栏 Ctrl+ \（反斜杠）

5、移动剪辑
Alt + 方向键 移动位置（每次一帧）
Alt + Shift + 方向键 移动位置（每次五帧）

6、抽取与删除
；（分号） 删除工作区片段
，（逗号） 抽取工作区片段

7、轨道宽度
Ctrl + 加减号 改变轨道宽度
Shift + 加减号 一次性改变所有轨道宽度

8、切换窗口
Shift + 1,2,3 快速切换窗口

9、自定义快捷键切换激活轨道
通过自定义快捷键Ctrl+1,2,3来切换激活视频轨道
通过自定义快捷键Ctrl+4,5,6来切换激活音频轨道
